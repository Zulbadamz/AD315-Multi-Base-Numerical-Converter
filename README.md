# Base Converter (Java Project)

## 📌 Description
This is a simple Java console application that converts numbers between different number systems (bases 2 to 16).

The program first converts the input number into decimal (base 10), then converts it into the desired target base.

---

## ⚙️ Features
- Supports number bases from 2 to 16
- Converts binary, octal, decimal, and hexadecimal numbers
- Handles invalid input errors

---

## 🧠 How it works
1. User enters a number
2. User enters the base of that number (2–16)
3. User enters the base to convert to (2–16)
4. Program:
   - Converts input → decimal
   - Converts decimal → target base
5. Displays result

---

## ▶️ How to run

### 1. Compile the program

javac BaseConverter.java

### 2. Run the program

javac BaseConverter
